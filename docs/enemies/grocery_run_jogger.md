# Grocery Run Jogger

*"Speed over quality."*

> — Flavor: "Too busy to ferment."

These frantic shoppers believe every second saved is worth sacrificing flavor.

They sprint through supermarket aisles collecting plastic-wrapped bread before anyone can recommend a local bakery.

Couriers dislike them immensely.

Fortunately, they cannot outrun the smell of fresh bread forever.
