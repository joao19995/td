# Pigeon with a Stolen Baguette

*"Crime is temporary. Bread is forever."*

> — Flavor: "Wanted in twelve bakeries."

No one knows why pigeons are so obsessed with stealing baguettes.

The Order has documented hundreds of incidents without reaching a conclusion.

Some birds eventually return the bread.

Others begin entire aerial bread-smuggling operations.

The stolen baguette is almost never industrial.

Which somehow makes everything even stranger.
