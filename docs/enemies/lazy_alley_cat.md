# Lazy Alley Cat

*"Fed by every bakery... especially the wrong ones."*

> — Flavor: "Powered entirely by stolen pastries."

Long ago, alley cats guarded artisan bakeries from rats.

This one discovered supermarket dumpsters.

Years of eating discarded industrial pastries have transformed it into an enormous loaf-shaped menace that refuses to move unless absolutely necessary.

The monks still insist it can be redeemed.

Mostly because they think it's adorable.
